# Little Obf Fallback
This is an obfuscation compatibility layer that implements a fallback for obfuscation, as obfuscation has been removed since version 26.1.
26.1以降で難読化が解除されたため、難読化のフォールバックを実装したobf互換レイヤーです。

It is only for mods that are based on MCPitanLib, but you can make it available for external mods with Config.
MCPitanLibを前提としているmodのみ対象ですが、Configで外部modも対象にすることができます。
